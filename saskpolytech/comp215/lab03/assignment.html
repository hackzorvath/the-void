Why 218px sometimes "works better"

The original question you included asked why margin-left:210px vs 218px. The difference comes down to extra pixels added by borders, padding or default body margins/scrollbars. If the left column has a 2px border and some padding, the effective visual space it takes is more than 210px. Using box-sizing:border-box prevents padding and border from increasing element width, which is the preferred fix. Increasing margin by a few pixels (e.g. to 218px) is a pragmatic workaround when you don't change the box model or when there are other layout quirks.

Student exercise idea

Try changing the values: set box-sizing off, add padding/borders, and see how the center column overlaps. Then toggle box-sizing:border-box back on and observe how the layout stabilizes.
